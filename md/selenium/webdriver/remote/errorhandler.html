处理 WebDriver 服务器返回的错误。

2.1. check_response(response)

检查来自 WebDriver 的 JSON 响应是否没有错误。

  • 参数：

    • response - 来自 WebDriver 服务器的 JSON 响应，作为字典对象。

  • Raises：

    如果响应包含错误消息。
